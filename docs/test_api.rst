==================
Test API Reference
==================

.. module:: moro8

moro8 contains multiple tests that are run with `CMocka <https://cmocka.org/>`_, and for
convenience, you can include the ``tests/moro8_testutils.h`` header to access some
utility functions to help you write your own tests.
